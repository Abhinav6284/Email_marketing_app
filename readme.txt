MARK EMAIL MARKETING APPLICATION
================================

SETUP INSTRUCTIONS:
1. Install Python 3.8+ if not already installed
2. Open terminal/command prompt in this folder
3. Install required packages: pip install -r requirements.txt
4. Run the application: python app.py
5. Open browser and go to: http://127.0.0.1:5000

FEATURES:
- User registration and login
- Contact management
- Email campaigns
- Campaign tracking and history
- SQLite database (auto-created on first run)

NOTES:
- Database will be created automatically in instance/users.db
- Configure your SMTP settings in smtp_cred.py for email sending
- Default access: http://127.0.0.1:5000
